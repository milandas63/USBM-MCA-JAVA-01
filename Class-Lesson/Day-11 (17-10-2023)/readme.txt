    =   Java GUI programming
        GUI stands for Graphical User Interface
        java.awt failed. The claim of Java, that it is platform independent was deceptive.
        So, javax.swing was introduced in Java version 1.2 onwards. The Java Swing project
        is known as JFC (Java Foundation Class).

        Swing provided a set of containers and components.
            AWT-CONTAINERS              AWT-COMPONENTS
            >   Frame                   >   Label
            >   Dialog                  >   TextField
            >   Panel                   >   Button
            >                           >   Checkbox
                                        >   MenuBar
                                        >   MenuItem
        
            SWING-CONTAINERS            SWING-COMPONENTS
            >   JFrame                  >   JLabel
            >   JDialog                 >   JTextField
            >   JPanel                  >   JButton
            >   JSplitPane              >   JCheckBox
                                        >   JMenuBar
                                        >   JMenu
                                        >   JMenuItem
